# Automated Corporate Workstation Setup (Ansible)

## 📌 Описание проекта

Данный проект представляет собой систему автоматизированной настройки корпоративных рабочих станций разработчиков с использованием Ansible.

Основная цель — создать воспроизводимую, безопасную и масштабируемую конфигурацию рабочих машин, которая может быть развернута в инфраструктуре компании с минимальным ручным вмешательством.

Проект ориентирован на поддержку нескольких Linux-дистрибутивов:

* Ubuntu 24.04
* Astra Linux Orel

---

## 🎯 Цели проекта

* Автоматизация настройки рабочих станций
* Обеспечение единообразия окружения
* Повышение уровня корпоративной безопасности
* Минимизация ручной настройки
* Поддержка мультиплатформенной инфраструктуры

---

## 🏗 Архитектура проекта

Проект построен на основе ролей Ansible:

* base — базовая настройка системы
* users — управление пользователями (планируется)
* security — базовые политики безопасности (планируется)
* docker — установка и настройка Docker (планируется)
* dev — инструменты разработки (планируется)

## ⚠️ Особенности Astra Linux

В Astra Linux используется отдельная логика установки пакетов:

* стандартный модуль apt Ansible не используется
* применяется raw из-за несовместимости python3-apt с кастомным Python
